# Titanic Data Analysis (Excel Project)

## Project Overview
This project analyzes the Titanic dataset to explore factors that influenced passenger survival.

## Tools Used
- Excel (Data Cleaning, Pivot Tables, Visualization)

## Data Cleaning
- Filled missing Age values using the average
- Filled missing Embarked values using the most frequent value
- Ignored Cabin column due to excessive missing data

## Key Analysis
1. Survival Rate by Gender  
   - Female passengers had a significantly higher survival rate than males  

2. Survival Rate by Passenger Class  
   - First-class passengers had the highest survival rate  
   - Third-class passengers had the lowest  

3. Age Distribution and Survival  
   - Younger passengers had relatively higher survival rates  

## Conclusion
Gender, passenger class, and age all played important roles in survival outcomes.
